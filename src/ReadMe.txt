
Libs for Environmental modeling 
1) auto_caline4: auto generation of the input for the input to the Caline4 moduels; 
2) ExtractImgs: efficient algorithms for extracting the large images; 
3) sqlLib: SQL programs for extracting spatial covariates such as distance to roadways, area proportion etc.;
4) tempporallyadjustedLib: programs for the temporally adjusted land-use estimates; 
5) webcrawl: programs for crawling the meteorological data and walkability.
